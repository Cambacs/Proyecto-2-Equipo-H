Introducción al sistema

El Monitor Ambiental es una plataforma web diseñada para visualizar, analizar y registrar datos de sensores ambientales. Los sensores miden temperatura, humedad y CO₂ de forma periódica y envían los datos a la base de datos.

Esta guía explica cómo usar cada sección de la web para que puedas aprovechar al máximo la información disponible.

Cómo navegar por la web

En el lado izquierdo de la pantalla encontrarás el panel de navegación. Desde ahí podés acceder a todas las secciones con un solo clic. El botón de modo claro/oscuro está al pie del panel.

  • Hacé clic en cualquier ítem del menú para ir a esa sección.
  • La sección activa aparece resaltada en el menú lateral.
  • Para salir del sistema, usá el botón "Salir del sistema" al pie del panel.

Sección: Último sensoreo

Muestra los datos más recientes registrados por el sensor. Se actualiza automáticamente con cada nuevo registro de la base de datos. Incluye:

  • Valores actuales de temperatura, humedad y CO₂.
  • Comparación con el sensoreo anterior (↑ empeoró, ↓ mejoró, → igual).
  • Recomendaciones automáticas según los valores detectados.
  • Temporizador de cuenta atrás para la próxima actualización.

Sección: Registros

La sección de Registros actuales muestra las últimas 50 lecturas del mes en curso. Podés filtrar por rango de fechas y por calidad del aire usando los controles de búsqueda.

Los Registros anteriores contienen el historial completo de los últimos 2 meses, también con filtros por fecha.

Configuración del Arduino

El sensor Arduino debe estar conectado a la misma red que el servidor. El microcontrolador envía los datos al servidor cada 30 minutos mediante una petición HTTP POST.

  • Conectar el Arduino a la red WiFi configurada.
  • Verificar que el LED verde del sensor esté encendido (estado activo).
  • Si el sensor no envía datos, reiniciarlo con el botón RESET.
  • Ante problemas de conexión, revisar la IP del servidor en el sketch.

EcoSense Technologies
Empresa dedicada al monitoreo ambiental inteligente. Desarrollamos soluciones IoT para medir, analizar y mejorar la calidad del aire en espacios de trabajo y estudio, con el objetivo de generar ambientes más saludables y productivos.
Objetivos del proyecto
  • Monitorear continuamente la calidad del ambiente de trabajo.
  • Alertar ante condiciones que afecten la salud o concentración.
  • Generar datos históricos para identificar patrones y tendencias.
  • Facilitar la toma de decisiones para mejorar el entorno.
